# 广告风格要求

## 概述
本文档定义不同的广告视频风格，用于指导混剪方案的风格化生成。

---

## 风格分类体系

### 风格维度
1. **叙事方式**：直接式、故事式、场景式
2. **节奏快慢**：快节奏、中等节奏、慢节奏
3. **情感基调**：理性、感性、幽默
4. **视觉风格**：简约、丰富、炫酷

---

## 一、快闪风格 (Fast Cuts)

### 核心特征
- **镜头时长**：1-2 秒/镜头
- **转场方式**：硬切为主
- **节奏**：紧凑、密集
- **音乐**：高 BPM（120-140）、电子、动感

### 适用场景
- 科技产品（手机、耳机、智能设备）
- 运动户外（运动服、装备）
- 年轻化品牌
- 双 11、618 等大促

### 剪辑要求
```
时长：15-30 秒
镜头数量：10-20 个
节奏：每拍切换 1 次
重点：
  - 产品多角度快速展示
  - 功能点快速罗列
  - 强烈的视觉冲击
```

### 叙事结构
```
爆点开场(2s) → 产品特性快闪(10s) →
使用场景闪回(10s) → 价格+CTA(3s)
```

### 禁忌
- ❌ 长镜头（>3 秒）
- ❌ 复杂叙事
- ❌ 过多文字说明

---

## 二、故事叙事风格 (Storytelling)

### 核心特征
- **镜头时长**：5-8 秒/镜头
- **转场方式**：淡入淡出、匹配切换
- **节奏**：舒缓、流畅
- **音乐**：叙事性强、情感化（钢琴、弦乐）

### 适用场景
- 高端产品（奢侈品、高端家电）
- 情感营销（母婴、礼品、健康）
- 品牌形象宣传
- 生活方式类产品

### 剪辑要求
```
时长：45-60 秒
镜头数量：8-12 个
节奏：每 4-8 拍切换 1 次
重点：
  - 完整的故事弧线
  - 人物情感表达
  - 产品自然融入场景
```

### 叙事结构
```
铺垫场景(10s) → 问题/需求(10s) →
产品登场(15s) → 解决方案(15s) → 情感升华(10s)
```

### 禁忌
- ❌ 生硬的产品展示
- ❌ 过于直白的促销信息
- ❌ 快节奏剪辑

---

## 三、简约直给风格 (Minimalist Direct)

### 核心特征
- **镜头时长**：3-5 秒/镜头
- **转场方式**：简单硬切
- **节奏**：中等、清晰
- **音乐**：轻快、简洁

### 适用场景
- 日用品（清洁、收纳、厨具）
- 功能性产品（工具、办公用品）
- 性价比产品
- 日常种草

### 剪辑要求
```
时长：20-30 秒
镜头数量：6-10 个
节奏：稳定中速
重点：
  - 产品清晰展示
  - 功能直观说明
  - 价格优势突出
```

### 叙事结构
```
产品特写(5s) → 功能演示(10s) →
使用效果(8s) → 价格+CTA(7s)
```

### 禁忌
- ❌ 复杂特效
- ❌ 过度包装
- ❌ 冗长铺垫

---

## 四、对比展示风格 (Before/After)

### 核心特征
- **镜头时长**：4-6 秒/镜头
- **转场方式**：分屏、滑动转场
- **节奏**：中等偏快
- **音乐**：节奏感强、正能量

### 适用场景
- 美妆护肤（前后对比）
- 清洁用品（效果对比）
- 减肥健身（身材对比）
- 家居改造（空间对比）

### 剪辑要求
```
时长：15-30 秒
镜头数量：8-12 个
节奏：对比节奏明确
重点：
  - 清晰的前后对比
  - 效果视觉化
  - 强调改变
```

### 叙事结构
```
问题场景(5s) → 产品使用(8s) →
效果对比(10s) → 用户反馈(4s) → CTA(3s)
```

### 禁忌
- ❌ 对比不明显
- ❌ 虚假夸大
- ❌ 过度美颜

---

## 五、教程演示风格 (Tutorial)

### 核心特征
- **镜头时长**：6-10 秒/镜头
- **转场方式**：步骤式切换
- **节奏**：中慢、清晰
- **音乐**：轻松、背景化

### 适用场景
- 美妆（化妆教程）
- 厨房家电（食谱演示）
- DIY 产品（制作过程）
- 电子产品（设置教程）

### 剪辑要求
```
时长：45-60 秒
镜头数量：6-10 个
节奏：每个步骤清晰
重点：
  - 步骤分解清晰
  - 操作可跟随
  - 结果可预期
```

### 叙事结构
```
成品展示(5s) → 步骤 1(10s) → 步骤 2(10s) →
步骤 3(10s) → 成品再现(8s) → 产品信息(7s)
```

### 禁忌
- ❌ 步骤跳跃
- ❌ 过快节奏
- ❌ 细节不清

---

## 六、UGC 风格 (User Generated Content)

### 核心特征
- **镜头时长**：3-6 秒/镜头
- **转场方式**：自然硬切
- **节奏**：真实、随性
- **音乐**：流行、BGM

### 适用场景
- 所有品类（强调真实体验）
- 口播种草
- 开箱测评
- 日常 vlog

### 剪辑要求
```
时长：30-60 秒
镜头数量：10-15 个
节奏：自然、不刻意
重点：
  - 真实感
  - 生活化场景
  - 口播自然
```

### 叙事结构
```
开场打招呼(3s) → 开箱/收到产品(10s) →
使用体验(30s) → 真实感受(10s) → 推荐(7s)
```

### 禁忌
- ❌ 过度剪辑
- ❌ 太专业化
- ❌ 明显广告感

---

## 七、情感共鸣风格 (Emotional)

### 核心特征
- **镜头时长**：8-12 秒/镜头
- **转场方式**：柔和转场、叠化
- **节奏**：舒缓、深情
- **音乐**：温暖、治愈系

### 适用场景
- 母婴产品
- 宠物用品
- 礼品类
- 公益营销

### 剪辑要求
```
时长：45-60 秒
镜头数量：6-8 个
节奏：情感铺垫充分
重点：
  - 情感细节捕捉
  - 人物表情特写
  - 温馨场景
```

### 叙事结构
```
日常场景(12s) → 情感触点(15s) →
产品融入(18s) → 情感升华(12s) → 品牌信息(3s)
```

### 禁忌
- ❌ 节奏过快
- ❌ 生硬产品植入
- ❌ 情感过载

---

## 八、炫技酷炫风格 (Stylish)

### 核心特征
- **镜头时长**：2-4 秒/镜头
- **转场方式**：特效转场、动态图形
- **节奏**：快速、动感
- **音乐**：电子、说唱、潮流

### 适用场景
- 潮牌服饰
- 游戏外设
- 潮玩手办
- 音乐设备

### 剪辑要求
```
时长：15-30 秒
镜头数量：12-20 个
节奏：强节奏感
重点：
  - 视觉特效
  - 色彩对比强烈
  - 动态元素多
```

### 叙事结构
```
视觉冲击开场(3s) → 产品炫技展示(15s) →
氛围渲染(9s) → 品牌 logo(3s)
```

### 禁忌
- ❌ 特效过载
- ❌ 看不清产品
- ❌ 眩晕感

---

## 九、促销紧迫风格 (Urgency)

### 核心特征
- **镜头时长**：2-3 秒/镜头
- **转场方式**：快速硬切
- **节奏**：急促、紧张
- **音乐**：紧迫感、倒计时

### 适用场景
- 限时促销
- 秒杀活动
- 清仓甩卖
- 大促预热

### 剪辑要求
```
时长：15-20 秒
镜头数量：8-12 个
节奏：极快
重点：
  - 价格突出
  - 倒计时元素
  - 优惠信息清晰
```

### 叙事结构
```
爆点优惠(3s) → 产品快闪(8s) →
倒计时(4s) → 疯狂 CTA(5s)
```

### 禁忌
- ❌ 信息不清晰
- ❌ 过度焦虑感
- ❌ 价格误导

---

## 十、高端品质风格 (Premium)

### 核心特征
- **镜头时长**：6-10 秒/镜头
- **转场方式**：优雅转场、慢动作
- **节奏**：从容、精致
- **音乐**：交响乐、爵士

### 适用场景
- 奢侈品
- 高端电子产品
- 精品咖啡/茶
- 高端家居

### 剪辑要求
```
时长：45-60 秒
镜头数量：6-8 个
节奏：舒缓、精致
重点：
  - 细节特写
  - 质感表现
  - 氛围营造
```

### 叙事结构
```
氛围铺垫(10s) → 产品细节(20s) →
使用场景(18s) → 品牌理念(12s)
```

### 禁忌
- ❌ 快节奏
- ❌ 促销信息
- ❌ 粗糙剪辑

---

## 风格选择决策树

```
产品类型
├─ 科技产品
│  ├─ 年轻用户 → 快闪风格 / 炫技风格
│  └─ 高端产品 → 高端品质风格
├─ 美妆护肤
│  ├─ 功效型 → 对比展示风格
│  ├─ 使用型 → 教程演示风格
│  └─ 高端线 → 高端品质风格
├─ 服装配饰
│  ├─ 潮牌 → 炫技风格 / UGC 风格
│  └─ 基础款 → 简约直给风格
├─ 食品饮料
│  ├─ 零食 → UGC 风格 / 快闪风格
│  └─ 高端 → 故事叙事风格
├─ 家居日用
│  ├─ 功能性 → 简约直给风格
│  ├─ 清洁类 → 对比展示风格
│  └─ 装饰类 → 情感共鸣风格
└─ 特殊活动
   ├─ 大促 → 促销紧迫风格
   └─ 品牌宣传 → 故事叙事风格
```

---

## 风格混搭建议

### 可以混搭
- **快闪 + 对比**：快速展示 + 效果对比
- **故事 + 情感**：叙事 + 情感升华
- **简约 + 教程**：直接展示 + 使用说明

### 不建议混搭
- ❌ 快闪 + 故事（节奏冲突）
- ❌ 高端 + 促销（调性冲突）
- ❌ 炫技 + 简约（风格冲突）

---

## 风格适配清单

在生成方案时，确保：

- [ ] 镜头时长符合风格要求
- [ ] 转场方式匹配风格
- [ ] 色调与风格一致

### 节奏层面
- [ ] 整体节奏符合定义
- [ ] 音乐风格匹配
- [ ] 剪辑点对齐节拍

### 内容层面
- [ ] 叙事方式符合风格
- [ ] 信息密度适配
- [ ] 情感基调统一
