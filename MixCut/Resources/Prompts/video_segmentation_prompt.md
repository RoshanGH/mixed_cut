## 切分原则（必须严格遵守）

### 原则1：语义完整性 ⭐️⭐️⭐️
- **不在句子中间切断**，每个片段必须是完整的语义单元
- 保留完整的排比、对比、因果等修辞结构
- 如果一段话包含完整的论证逻辑（问题→原因→解决），则作为一个整体
- 允许片段较长（10-15秒），只要语义需要完整表达

**正例**：
```
✓ "如果头有头臭那就跟着康王走，如果头皮雪花飘飘还是跟着康王走，
   如果头皮长小包包更要跟着康王走，拒绝头皮问题跟着康王走就对了"
   → 完整的排比结构，14.48秒，保留完整
```

**反例**：
```
✗ "如果头有头臭那就跟着康王走，如果头皮雪花飘飘..." [切断]
   → 破坏了排比结构，语义不完整
```

### 原则2：主题一致性 ⭐️⭐️⭐️
- 相同主题、相同论述对象的内容归为一个片段
- 即使有轻微停顿或语气词，如果主题一致，仍保持为同一片段
- 当主题切换时（如从"痛点描述"转向"产品方案"），则切分新片段

**判断标准**：
```
主题一致 → 合并
- "头皮出油、头皮发痒、头皮有包" → 都是痛点描述
- "添加了二硫化硒、复配了角鲨烷、含有神经酰胺" → 都是成分介绍

主题切换 → 切分
- 从"你的头发一天不洗就油" → 转到"康王洗发水含有二硫化硒"
  (痛点 → 产品方案)
```

### 原则3：数据驱动 ⭐️⭐️⭐️⭐️⭐️
- **优先标记高互动片段**：点击率 > 75% 或留存率高的时间段
- 低流失片段（< 10%流失率）应保持完整，不要过度切割
- 互动峰值时间段即使内容复杂，也尽量保持为独立片段
- 高流失时间段可以适度细分，便于后续优化替换

**数据指标映射**：
```
点击率 > 75%     → 质量评分 9.0-9.5，标记为"开头"或"噱头引入"
流失率 < 10%     → 质量评分 8.5-9.0，标记为核心内容
留存率回升区间   → 质量评分 8.0-8.5，适合作为转折点
多次互动峰值     → 标记为"高质量"，优先用于混剪
```

### 原则4：关键帧对齐 ⭐️⭐️
- 片段的起止时间点**尽量对齐到关键帧**（场景切换点）
- 如果关键帧时间点与语义边界不一致，**优先保证语义完整性**
- 允许在关键帧前后 ±0.5 秒范围内调整，确保语义自然

**调整策略**：
```
语义边界 @ 10.3s，最近关键帧 @ 10.8s
→ 优先选择 10.3s（语义完整性 > 关键帧对齐）

语义边界 @ 10.3s，最近关键帧 @ 10.1s
→ 可选择 10.1s（关键帧对齐，且误差小）
```

## 切分粒度标准

### 时长分布目标（黄金比例）
```
短镜头 (≤5s)     : 10-15%  → 用于爆点、重点强调
标准镜头 (5-10s)  : 40-50%  → 核心镜头，信息传递 ⭐️ 主力
长镜头 (10-15s)   : 30-35%  → 完整叙述、情感铺垫
超长镜头 (>15s)   : 5-10%   → 完整故事段落（尽量避免）
```

### 自适应粒度策略
```
视频总时长 < 40s  → 平均粒度 5-8s/片段 （快节奏，灵活组合）
视频总时长 40-80s → 平均粒度 8-12s/片段（均衡型）
视频总时长 > 80s  → 平均粒度 10-15s/片段（长叙述，保留完整性）
```

## 片段类型分类标准

### 必须使用以下标准类型（8种核心类型）

#### 1. 噱头引入 (Hook)
**识别特征**：
- 前3-10秒的开场内容
- 反常识观点、惊人数据、强烈情绪
- 反向劝退、问题质疑
- 视觉冲击、快速节奏

**时长范围**：2-5秒（可达10秒如果内容丰富）
**位置标注**：开头
**质量评分参考**：8.5-9.5（开场非常重要）

**示例**：
- "过年回家不想频繁洗头就用康王二硫化硒洗发水"
- "给大家示范一下洗发水的错误用法"

---

#### 2. 痛点 (Pain Point)
**识别特征**：
- 场景化描述用户问题
- 细节化痛点（不是泛泛而谈）
- 引导用户自我识别（"如果你也..."）
- 强调问题的严重性、影响

**时长范围**：5-15秒（痛点描述需要细节）
**位置标注**：开头或中间
**质量评分参考**：8.5-9.5（痛点是转化的核心驱动）

**示例**：
- "头发一天不洗就无名的痒，晚上躺下睡觉时就更痒，有时候感觉像蚂蚁在爬..."
- "头皮问题头上有包包点点一挠还有白泥"

---

#### 3. 产品方案 (Solution)
**识别特征**：
- 产品核心功能介绍
- 成分/技术/工艺讲解
- 与普通产品的对比
- 产品分类和适用场景

**时长范围**：8-20秒（需要充分展开）
**位置标注**：中间
**质量评分参考**：8.0-9.0

**示例**：
- "世界上只有两种人还在天天洗头...里面添加了1%二硫化硒，复配了角鲨烷和神经酰胺..."
- "这个适合头油有味用，头屑头痒用，秋冬头皮有疙瘩搭搭用这一瓶就够了"

---

#### 4. 效果展示 (Results)
**识别特征**：
- 前后对比、感官描述
- 生动比喻、具象化效果
- 用户使用后的变化
- 即时效果或长期效果

**时长范围**：5-12秒
**位置标注**：中间
**质量评分参考**：8.0-9.0

**示例**：
- "不管你是头发终日出油还是头皮屑满天飞，你试一下它...既有里子又有面子"
- "洗了能从油扁塌变蓬松...持久蓬松，护头皮效果也很好"

---

#### 5. 信任背书 (Social Proof)
**识别特征**：
- 用户见证、评价
- KOL/专家推荐
- 品牌历史、资质
- 销量数据、获奖信息

**时长范围**：5-12秒
**位置标注**：中间
**质量评分参考**：7.5-8.5

**示例**：
- "老牌子国货，值得信赖"
- "已经卖出XXX万瓶"

---

#### 6. 价格对比 (Price Anchor)
**识别特征**：
- 与竞品价格对比
- 原价 vs 现价
- 性价比强调
- 价格锚点设置

**时长范围**：3-8秒（简洁有力）
**位置标注**：中间或结尾
**质量评分参考**：8.0-9.0

**示例**：
- "如果有花99一瓶买的洗发水还没到货的话那你赶紧退了吧"
- "现在直播间有活动..."

---

#### 7. 活动福利 (Promotion)
**识别特征**：
- 限时优惠、折扣信息
- 买赠套装、满减活动
- 稀缺性强调（库存、时间）
- 超值感堆叠

**时长范围**：5-20秒（福利堆叠需要时间）
**位置标注**：结尾
**质量评分参考**：8.5-9.5（促销是转化关键）

**示例**：
- "买一瓶清爽款正装我带你送一瓶清爽款正装，再给你送两瓶丰盈款正装..."
- "现在直播间还有更多福利赶紧来看看吧"

---

#### 8. 行动号召 (Call-to-Action)
**识别特征**：
- 直接购买指令
- 紧迫感制造
- 劝阻竞品/理性劝退
- 进直播间、下单等明确行动

**时长范围**：3-10秒
**位置标注**：结尾
**质量评分参考**：8.5-9.5（CTA 是转化的最后一步）

**示例**：
- "趁现在年终大促半价赶紧进直播间呀"
- "有头皮问题的姐妹真的可以闭眼入"

---

### 可选类型（根据实际内容使用）

#### 9. 产品定位 (Product Positioning)
**识别特征**：
- 明确产品适用人群
- 产品多维度分类
- 使用场景说明

**时长范围**：3-8秒
**位置标注**：开头

**示例**：
- "这个适合头油有味用，头屑头痒用，秋冬头皮有疙瘩搭搭用"

---

#### 10. 产品使用教育 (Usage Education)
**识别特征**：
- 使用方法说明
- 使用频率建议
- 注意事项提示

**时长范围**：5-15秒
**位置标注**：中间

## 位置标注规则

每个片段必须标注其在原视频中的**逻辑位置**（不是时间位置）：

### "开头" (Opening)
**判断标准**：
- 视频前 20% 时长内的片段
- 承担"吸引注意力"功能的内容
- 通常是：噱头引入、开场痛点、产品定位

**特性**：
- 有明显的"开场感"（建立悬念、快速吸引）
- 混剪时应优先放在新视频的开头位置

---

### "中间" (Middle)
**判断标准**：
- 视频 20%-80% 时长的片段
- 承担"信息传递、论证说服"功能
- 通常是：产品方案、效果展示、信任背书、价格对比

**特性**：
- 核心内容，没有强烈的位置属性
- 混剪时可以灵活排列，只需保证语义连贯

---

### "结尾" (Ending)
**判断标准**：
- 视频后 20% 时长的片段
- 承担"驱动转化、行动号召"功能
- 通常是：活动福利、行动号召、紧迫感制造

**特性**：
- 有明显的"收尾感"（总结、催促、制造紧迫）
- 混剪时应优先放在新视频的结尾位置

## 质量评分标准

为每个片段评分（0-10分），考虑以下因素：

### 评分维度

#### 1. 数据表现（权重 40%）
- 点击率 > 80%：+2 分
- 流失率 < 10%：+2 分
- 留存率稳定或上升：+1 分
- 互动峰值时段：+1 分

#### 2. 内容质量（权重 30%）
- 语义完整、逻辑清晰：+1.5 分
- 信息密度高、表达生动：+1 分
- 有明确的功能（痛点/方案/CTA）：+0.5 分

#### 3. 混剪价值（权重 30%）
- 时长适中（5-10秒）：+1.5 分
- 可独立理解（不依赖上下文）：+1 分
- 视觉吸引力强：+0.5 分

### 评分等级
```
9.0-10.0：顶级片段，必选
8.5-8.9： 优秀片段，优先选择
8.0-8.4： 良好片段，常规使用
7.5-7.9： 可用片段，按需使用
< 7.5：   质量不足，避免使用
```

## 关键词提取规则

为每个片段提取 3-6 个关键词：

### 提取优先级
1. **产品名称/品牌**：康王、二硫化硒
2. **核心痛点**：头油、头痒、头屑、雪花
3. **功能成分**：角鲨烷、神经酰胺
4. **效果描述**：蓬松、清爽、去屑
5. **促销信息**：买一送一、半价、活动

### 关键词要求
- 简洁（1-4个字）
- 高频（在文本中重复出现）
- 有区分度（能代表这个片段的特性）

## 验证清单（生成后自检）

在完成切分后，请逐一检查：

### ✅ 语义完整性检查
- [ ] 每个片段的文本都是完整的句子或段落
- [ ] 没有在排比、对比、因果结构中间切断
- [ ] 片段之间没有重叠或遗漏

### ✅ 类型分类检查
- [ ] 每个片段都有明确的类型标注
- [ ] 类型分布合理（痛点、产品方案、CTA 都有）
- [ ] 开头有"噱头引入"或"痛点"
- [ ] 结尾有"行动号召"

### ✅ 位置标注检查
- [ ] "开头"片段占比 15-25%
- [ ] "中间"片段占比 45-55%
- [ ] "结尾"片段占比 25-35%

### ✅ 时长分布检查
- [ ] 标准镜头（5-10秒）占比 40-50%
- [ ] 长镜头（10-15秒）占比 30-35%
- [ ] 避免过多超长镜头（>20秒）

### ✅ 质量评分检查
- [ ] 每个片段都有质量评分（0-10）
- [ ] 平均分数 ≥ 8.0
- [ ] 高质量片段（≥8.5）占比 ≥ 50%

### ✅ 数据驱动检查
- [ ] 高点击率时段被标记为高质量
- [ ] 高流失时段被适度细分或降低评分
- [ ] 质量评分的 reasoning 包含数据依据

## 常见错误及纠正

### ❌ 错误1：过度切割
**表现**：将完整的论述切成多个 3-5 秒的短片段

**纠正**：
- 检查是否破坏了语义完整性
- 相同主题的内容应合并
- 宁可片段长一些（10-15秒），也要保证完整

---

### ❌ 错误2：位置标注不准确
**表现**：把"行动号召"标记为"开头"

**纠正**：
- 根据片段的功能判断位置，而非时间位置
- "行动号召"必须是"结尾"
- "噱头引入"必须是"开头"

---

### ❌ 错误3：类型分类模糊
**表现**：无法区分"产品方案"和"效果展示"

**判断标准**：
- "产品方案"：讲产品是什么、有什么成分、如何工作
- "效果展示"：讲用了之后会怎样、有什么变化

---

### ❌ 错误4：质量评分缺乏依据
**表现**：所有片段都是 8.0 分

**纠正**：
- 必须基于数据指标评分
- 高点击率/低流失率的片段应该 ≥ 9.0
- 质量评分的差异应该明显（7.5-9.5）

## 进阶优化建议

### 1. 考虑混剪组合可行性
- 每种类型至少保留 2-3 个片段（便于多样化组合）
- "痛点"类片段可以适度多切（不同痛点场景）
- "行动号召"类片段保留多个版本（不同紧迫度）

### 2. 标注片段间的衔接关系
在 JSON 中可增加字段：
```json
"transition_to_next": "自然" / "需要转场" / "主题跳跃"
```

### 3. 标注视觉特征
如果有视觉分析，可增加：
```json
"visual_features": {
  "scene_type": "产品特写" / "人物口播" / "效果对比",
  "color_tone": "暖色" / "冷色",
  "motion_intensity": "静态" / "动态"
}
```
