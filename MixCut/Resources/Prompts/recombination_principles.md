# 混剪原则

## 概述
本文档定义视频混剪的核心原则，用于指导 AI 生成高质量的混剪方案。

---

## 一、叙事结构原则

### 1.1 经典三段式
```
吸引注意 → 建立兴趣 → 促成行动
(Hook)     (Interest)   (Action)
```

**应用：**
- 15 秒视频：Hook(5s) + Interest(5s) + Action(5s)
- 30 秒视频：Hook(8s) + Interest(15s) + Action(7s)
- 60 秒视频：Hook(10s) + Interest(40s) + Action(10s)

### 1.2 AIDA 模型
```
Attention（注意） → Interest（兴趣） →
Desire（欲望）   → Action（行动）
```

**片段映射：**
- **Attention**：开场片段（悬念、冲突、惊喜）
- **Interest**：产品介绍 + 使用场景
- **Desire**：效果展示 + 用户评价
- **Action**：优惠信息 + 行动号召

### 1.3 故事弧线
```
铺垫 → 冲突/问题 → 解决方案 → 结果 → 号召
```

**适用场景：** 长视频（45-60 秒），强调情感连接

---

## 二、时长控制原则

### 2.1 平台最佳实践

| 平台 | 推荐时长 | 黄金时长 | 备注 |
|------|---------|---------|------|
| 抖音 | 15-30s | 21s | 完播率优先 |
| 快手 | 30-60s | 45s | 深度内容 |
| 小红书 | 15-60s | 30s | 种草场景 |
| 视频号 | 30-60s | 50s | 微信生态 |

### 2.2 时长分配策略

**15 秒方案：**
- 开场：3 秒（20%）
- 核心内容：9 秒（60%）
- 行动号召：3 秒（20%）

**30 秒方案：**
- 开场：5 秒（17%）
- 产品介绍：8 秒（27%）
- 使用场景：10 秒（33%）
- 结尾：7 秒（23%）

**60 秒方案：**
- 开场：6 秒（10%）
- 产品介绍 + 使用场景：30 秒（50%）
- 效果展示 + 评价：15 秒（25%）
- 优惠 + 行动号召：9 秒（15%）

---

## 三、节奏控制原则

### 3.1 快慢节奏

**快节奏（1-2 秒/镜头）：**
- 适用：开场、产品特点罗列、快闪效果
- 目标：营造紧迫感、吸引注意

**中等节奏（3-5 秒/镜头）：**
- 适用：使用场景、效果展示
- 目标：信息传递、理解消化

**慢节奏（5-8 秒/镜头）：**
- 适用：情感故事、细节展示
- 目标：情感共鸣、沉浸体验

### 3.2 节奏变化曲线

```
开场（快） → 介绍（中） → 场景（中慢） →
高潮（快） → 结尾（中快）
```

**原则：** 避免单一节奏，保持节奏变化

---

## 四、转场规则

### 4.1 转场类型

1. **硬切（Cut）**
   - 最常用，适合快节奏
   - 用于：相似场景、同类型片段

2. **淡入淡出（Fade）**
   - 时间跨度转换
   - 用于：段落转换、情绪过渡

3. **关键帧对齐**
   - 在场景切换点转场
   - 用于：保持视觉连贯性

4. **动作匹配**
   - 通过相似动作连接
   - 用于：流畅的叙事

### 4.2 转场时机

- **优先使用**：场景检测的关键帧
- **避免在**：说话中途、动作过程中
- **特殊处理**：音乐节拍点、情感高潮点

---

## 五、内容匹配原则

### 5.1 音画同步

- **视觉与语音**：画面与解说词内容一致
- **视觉与文字**：屏幕文字与画面匹配
- **动作与音效**：关键动作与声音同步

### 5.2 情感连贯性

**正向情感：** 快乐 → 兴奋 → 满足
**中性情感：** 好奇 → 理解 → 认同
**负向转正：** 困扰 → 解决 → 满意

**原则：** 避免情感跳跃过大

### 5.3 信息密度

- **高密度**（15-20 秒）：每秒 1 个信息点
- **中密度**（30-45 秒）：每 2-3 秒 1 个信息点
- **低密度**（60 秒+）：每 3-5 秒 1 个信息点

---

## 六、素材选择原则

### 6.1 质量优先级

1. **画面质量**：清晰度、构图、光线
2. **内容相关性**：与产品/主题的匹配度
3. **情感强度**：能否引发观众共鸣
4. **信息价值**：包含的有效信息量

### 6.2 去重原则

- **同一场景**：不超过 2 次使用
- **相似镜头**：间隔至少 10 秒
- **重复话术**：避免完全重复

### 6.3 多样性

**视觉多样性：**
- 远景、中景、近景、特写交替
- 不同角度（正面、侧面、俯视、仰视）
- 不同场景（室内、室外、工作、休闲）

**内容多样性：**
- 产品 + 人物 + 场景
- 静态 + 动态
- 实拍 + 特效

---

## 七、音乐匹配原则

### 7.1 风格匹配

| 产品类型 | 推荐音乐风格 | 节奏 |
|---------|------------|-----|
| 科技产品 | 电子、现代 | 中快 |
| 美妆护肤 | 轻快、优雅 | 中等 |
| 食品饮料 | 活泼、温暖 | 中快 |
| 服装配饰 | 时尚、节奏感 | 快 |
| 家居用品 | 温馨、舒缓 | 慢 |

### 7.2 节拍对齐

- **关键镜头切换**：对齐音乐重拍（每 4 拍）
- **高潮片段**：对齐音乐高潮段落
- **结尾**：与音乐结束点同步

---

## 八、特殊场景处理

### 8.1 多产品混剪

**策略 1：并列式**
```
开场 → 产品 A(10s) → 产品 B(10s) → 产品 C(10s) → 结尾
```

**策略 2：穿插式**
```
开场 → A 介绍 → B 介绍 → A 场景 → C 介绍 → B 场景 → C 场景 → 结尾
```

**原则：**
- 每个产品至少 8 秒曝光
- 保持产品间平衡
- 可按销量/重要性分配时长

### 8.2 系列视频

**一致性要求：**
- 开场风格一致（模板化）
- 结尾行动号召一致
- 整体色调和节奏一致

**差异化要求：**
- 核心内容不同
- 素材不重复
- 突出各自卖点

### 8.3 季节性/节日主题

- **春节**：喜庆、团圆、红色基调
- **双 11**：促销、紧迫感、快节奏
- **夏季**：清爽、活力、明亮色彩
- **冬季**：温暖、舒适、暖色调

## 九、质量检查清单

### 9.1 结构检查
- [ ] 是否有明确的开场、发展、结尾？
- [ ] 叙事逻辑是否清晰？
- [ ] 时长分配是否合理？

### 9.2 内容检查
- [ ] 核心卖点是否突出？
- [ ] 是否有明确的行动号召？
- [ ] 信息密度是否适中？

### 9.3 技术检查
- [ ] 转场是否自然？
- [ ] 音画是否同步？
- [ ] 画面质量是否统一？

### 9.4 情感检查
- [ ] 情感基调是否连贯？
- [ ] 是否能引发目标情感？
- [ ] 结尾是否有情感高点？

## 十、优化迭代原则

### 10.1 A/B 测试维度

1. **时长**：15s vs 30s
2. **开场方式**：悬念 vs 直接展示
3. **节奏**：快节奏 vs 中等节奏
4. **结尾**：优惠强调 vs 产品强调

### 10.2 数据驱动优化

**关键指标：**
- 完播率：3 秒、5 秒、完整
- 点击率：购物车、链接
- 转化率：下单、加购

**优化策略：**
- 完播率低 → 缩短时长、加强开场
- 点击率低 → 强化行动号召、优惠信息
- 转化率低 → 增强信任背书、用户评价

---

## 十一、禁忌规则

### 11.1 内容禁忌
- ❌ 虚假夸大宣传
- ❌ 误导性对比
- ❌ 侵权素材使用
- ❌ 违禁词汇

### 11.2 技术禁忌
- ❌ 过度转场特效（眼花缭乱）
- ❌ 音量忽大忽小
- ❌ 画面抖动、模糊
- ❌ 文字遮挡关键内容

### 11.3 体验禁忌
- ❌ 信息过载（说太快、太多）
- ❌ 单调重复
- ❌ 没有重点
- ❌ 结尾草率
